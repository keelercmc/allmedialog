## Installation

### Requirements
- [Node.js](https://nodejs.org/en/)

### Clone
```shell
$ git clone https://github.com/keelercmc/allmedialog
```

### Setup
```shell
$ npm install
```

### Run
```shell
$ npm start
```

### Test
```shell
$ npm test
```

### Run Build
```shell
$ npm run build
```

### Eject
```shell
$ npm eject
```
